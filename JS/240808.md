### 템플릿 리터럴

backtick(`)을 사용해서 문자열을 표현한 것을 템플릿 리터럴이라고 한다.

백틱 내에서 작성하면 줄바꿈을 쉽게 할 수 있고, 문자열 내부에 표현식을 포함할 수 있게 된다.

```js
// ES6
console.log(`string text line1
string text line2`);

const a = 5;
const b = 10;

//ES5
console.log("fifteen is" + (a + b) + " and\nnot " + (2 * a + b) + ".");
// "fifteen is 15 and
// not 20."

// ES6
console.log(`fifteen is ${a + b} and
not ${2 * a + b}.`);
// "fifteen is 15 and
// not 20."
```
